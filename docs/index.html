Everything you need

A focused toolset for pairing audio with imagery.

Easy-to-use GUI

A clean tkinter interface handles everything — no terminal required for day-to-day use.

Multiple Audio Formats

Works with WAV, MP3, M4A, AAC, FLAC, OGG, and WMA — whatever your workflow produces.

Multiple Image Formats

Accepts JPG, PNG, TIFF, BMP, and GIF. Images are automatically scaled and padded to 720p.

Batch Processing

Select a whole folder of audio files and assign images to them simultaneously.

YouTube-Ready 720p Output

Exports H.264/AAC MP4 files at 1280×720 with yuv420p pixel format for maximum compatibility.

Combined Export

Merge all videos into a single file with a 1-second black transition and subtle audio click between clips.

Requirements

Pre-built binaries only need ffmpeg. Running from source requires Python too.

ffmpeg Required for all platforms

SlideForge calls ffmpeg under the hood. It must be on your system PATH.

macOS 
brew install ffmpeg
Ubuntu / Debian 
sudo apt install ffmpeg
Windows

Download the installer from ffmpeg.org and add it to your PATH.

Python 3.6+ Source only

Only needed when running directly from source. Pre-built binaries include the Python runtime.

All Python dependencies are part of the standard library (tkinter, subprocess, threading, pathlib) — no pip install step needed.

How to Use SlideForge

From launch to exported video in a few clicks.

01

Select your audio folder

Click Select Folder to open a folder containing your audio files. SlideForge will list all supported audio files it finds.

02

Associate images with audio files

You have three ways to assign an image to each audio file:

  • Double-click an audio file in the list to pick its image.
  • Select a file, then click Select Image for Selected File.
  • Click Select Images for All to step through every file one by one.
03

Create videos

Click Create Videos, choose an output folder, and SlideForge will generate a 720p MP4 for each audio-image pair using ffmpeg.

04

Export a combined video (optional)

After creating two or more videos, Export Combined Video becomes active. Click it to concatenate all videos into a single file, separated by a 1-second black screen with a subtle audio click.
